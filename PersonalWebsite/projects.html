2048 Clone (Love2D, lua)

Bullet Loop (P5js, Javascript)

(link) A game created for the Ludum Dare 47 game jam with two other teammates under the theme "Stuck in a Loop". The game is a simple bullet hell shooter where the bullets loop around the stage. i.e. If you shoot a bullet to the right side of the stage, the bullet will reappear on the left side of the stage. The game was create with P5js.

Clueless (Love2D, lua)

(link) A game made for the 2021 Global Game Jam. This game was also created in 48 hours under the theme of "Lost & Found". Amongst several NPCs you are tasked in finding a "murderer" by interrogating the NPCs. Each time the game is started there is a random murderer and each NPCs have new dialogue options.

Out on a Limb (Love2D, lua)

(link) A game created for the 2020 Global Game Jam that I worked on with some friends. The game was created in 48 hours under the theme "repair". A simple platformer where the a robot collects its body parts across the map and gain abilities for each limb it collects.

Pictochat Twitter (Py4Web)

(link) A project for CSE-183: Web Applications that I worked on with one other teammate. It takes the "pictochat" feature from the Nintendo DS where users could send small doodles to other people within their vacinity. We took this concept and created a website with the twitter format where users can upload small doodles and share them with the people that follow them.
